# DaemonMC Plugin House

**Welcome to DaemonMC Plugin House! this is a repository made for plugins for [DameonMC Server Software](https://github.com/TeamDeamonMC/DaemonMC)**

>[!Important]
>Some Plugins on this Page might be outdated with v1 .NET8 Framework Make sure you update all plugins in **CSPROJ** to .NET10 or latest

>[!Warning]
>We are no-longer going to be using .NET8 as of April 14th 2026 after that V2 is ready to release to the public on .NET10 for C# any modified version of v1 on the .NET8 Framework will not be supported by the team

>[!Caution]
>Please Make sure you are aware of all updates on the [Server Software](https://github.com/TeamDeamonMC/DaemonMC)
